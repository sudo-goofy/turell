# RGB Hex Puzzle

Eine interaktive Web-App zum Erlernen von **Hexadezimal-Farbcodes** im Informatikunterricht, inspiriert von den Lichtinstallationen des Künstlers James Turrell.

## Live Demo

> Nach dem Hosting auf GitHub Pages hier eintragen:  
> `https://sudo-goofy.github.io/turell/`

## Über das Projekt

Drei leuchtende Licht-Orbs (Rot, Grün, Blau) schweben im Dunkeln. Durch Ziehen der Orbs über den Bildschirm mischen sich die Farben additiv – genau wie Licht in der Realität. Ein Hex-Keypad unten zeigt den resultierenden `#RRGGBB`-Farbcode an und lädt dazu ein, bestimmte Zielfarben zu erraten und einzutippen.

**Lernziele:**
- Verstehen des RGB-Farbmodells und additiver Farbmischung
- Aufbau von Hexadezimal-Farbcodes (`#RRGGBB`)
- Umrechnung zwischen Dezimal (0–255) und Hex (00–FF)

## Verwendung

Da die App aus einer einzigen HTML-Datei besteht, gibt es keine Installation oder Build-Schritte.

1. `index.html` in einem Browser öffnen – fertig.
2. Oder: Live auf GitHub Pages besuchen (Link oben).

## Technologie

- Reines HTML / CSS / JavaScript – keine externen Abhängigkeiten
- Touch- und Maus-Unterstützung
- Responsive Design (funktioniert auf Tablet und Desktop)
